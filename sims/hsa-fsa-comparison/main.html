About FICA Savings FICA savings (7.65% = 6.2% Social Security + 1.45% Medicare) apply only when contributions are made through payroll deduction. Self-employed HSA contributions save federal and state income tax but not FICA. If you earn above $176,100, Social Security (6.2%) no longer applies to wages above that amount.
