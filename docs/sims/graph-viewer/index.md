## How to Use

- **Search**: Type in the search box to find specific concepts (e.g., "W-2", "deduction", "capital gains")
- **Filter by Category**: Use the checkboxes in the sidebar to show/hide topic groups
- **Navigate**: Drag to pan, scroll to zoom, click a node to highlight its connections
- **Statistics**: The sidebar shows live counts of visible nodes and edges

## Graph Structure

Arrows point **from** a concept **to** its prerequisites — follow the arrows backward to see what you need to learn first.

- **Box-shaped nodes** — foundational concepts (no prerequisites)
- **Rounded nodes** — concepts that build on others
